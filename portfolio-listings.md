## Fiverr — Portfolio item (separate from the Gig)

For new sellers this is load-bearing social proof. **Made on Fiverr: OFF** (this is a concept project, not Fiverr-delivered work).

### Title

> Storestack — App marketplace for e-commerce merchants (concept)

### Description (≈200 words)

> A full-stack app marketplace built as a portfolio case study. Storestack is a platform-agnostic marketplace where independent e-commerce merchants (WooCommerce, Magento, custom) can browse and install apps that extend their stores, developers can publish and manage their listings, and admins moderate the queue end-to-end.
>
> The build covers everything I deliver on the gig: custom session auth (jose-signed JWT in an httpOnly cookie, no Auth.js dependency), Server Actions throughout, a 15-model Prisma schema on Neon Postgres, transactional email via Resend (sign-up verify, password reset, install confirmation, developer approve/reject), Markdown rendering with HTML sanitizer, full-text search across name + description + developer, role-based middleware (Next 16 Proxy), and an immutable audit log for every privileged action.
>
> Notable hard problems solved: Prisma 7's required driver adapter, Vercel's serverless function freeze killing fire-and-forget email, and React 19 form integration via `useActionState`.
>
> Concept project — not real client work. Storestack is fictional.
